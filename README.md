# Odin Recipes

A simple recipe website built as part of The Odin Project Foundations curriculum.

## About

This project is a basic multi-page website that showcases several recipes. It was created to practice fundamental HTML concepts such as page structure, links, images, lists, and project organization.

## Features

* Homepage containing links to all recipes
* Individual recipe pages
* Recipe images
* Ingredients lists
* Step-by-step cooking instructions
* Navigation back to the homepage

## Recipes Included

* Chana Masala
* Dal Makhani
* Paneer Butter Masala
* Ven Pongal

## Technologies Used

* HTML5
* Git
* GitHub

## What I Learned

Through this project, I learned:

* How to structure a website using HTML
* How to create and organize files and folders
* How to use relative links between pages
* How to add images to web pages
* How to create ordered and unordered lists
* How to track changes with Git
* How to publish projects using GitHub
